## 2. المهام المنجزة

### NAV-002 — Dashboard Stats Company-Scoped
- إصلاح `/api/dashboard/stats` لتصفية البيانات حسب `companyId` للمستخدم الحالي
- إضافة `totalSuppliers` إلى الإحصائيات
- تحديث صفحة `/dashboard` لعرض 5 بطاقات إحصائية (عملاء، موردين، مواد، فواتير، مستخدمين)
- إضافة رابط "المشتريات" إلى الإجراءات السريعة

### NAV-006 — Inventory Hub Page
- إنشاء `/dashboard/inventory/page.tsx` كصفحة رئيسية للمخزون
- روابط لجميع وحدات المخزون: المخازن، الأرصدة، الحركات، التسويات، التحويلات، التقييم، التدقيق
- إضافة رابط "المخزون" في الشريط الجانبي

### NAV-004 — Reports Page Enhancement
- تحسين صفحة التقارير من placeholder إلى صفحة تحتوي روابط لتقارير موجودة: تقييم المخزون، تدقيق المخزون، أرصدة المخزن، سجل النشاطات
- إضافة رابط "التقارير" في الشريط الجانبي

### NAV-005 — Settings Page Enhancement
- تحسين صفحة الإعدادات من placeholder إلى صفحة تحتوي روابط لإعدادات النظام: الشركات، الفروع، المستخدمين، الأدوار، الفترات المالية، سعر الصرف، حسابات الدفع، دليل الحسابات
- إضافة رابط "الإعدادات" في الشريط الجانبي
